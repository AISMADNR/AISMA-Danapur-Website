AISMA Structure

  • Central Body: Governs all Indian Railways' Station Masters under AISMA.
  • Zonal Bodies (10+):
    • Each zone (e.g., NR, ER, SR, CR, WR, etc.) has its own executive and finance team.
    • Divisional Bodies: Under each zone, several divisions like Delhi, Mumbai, Chennai, etc.
      • Branch Bodies: Each division contains several branches, usually per station cluster.
